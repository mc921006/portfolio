# AI Language Study — AI 기반 다국어 학습 서비스

## 프로젝트 개요

AI를 활용하여 영어, 베트남어, 일본어를 보다 효율적으로 학습할 수 있도록 개발한 개인 학습 서비스입니다.

언어별 학습 콘텐츠를 하나의 서비스에서 제공하며, AI를 활용하여 학습 데이터를 생성하고 회화 질문 및 피드백을 제공합니다.

영어는 CEFR(A1~C2), 일본어는 JLPT(N5~N1), 베트남어는 실생활 중심의 단어 학습을 지원하며, 일본어는 히라가나와 카타카나 학습 기능도 함께 제공합니다.

Web Speech API를 활용한 발음(TTS), PWA(App 설치), 반응형 UI를 적용하여 Desktop과 Mobile 환경 모두에서 사용할 수 있도록 구현했습니다.

- 영어 CEFR(A1~C2)

- 단어 의미 제공

- 예문 제공
- 발음(TTS) 지원

### 구현 포인트

## 트러블 슈팅

### AI 생성 데이터 중복 문제

### 문제

AI가 생성한 단어가 기존 데이터와 중복되어 저장되는 문제가 발생했습니다.

### 해결

- language 기준 중복 체크
- 기존 단어 유지
- 새로운 단어만 저장
- 공통 AI Seed 스크립트에 중복 검증 로직 추가

---

### AI 회화 UX 개선

### 문제

답변 평가와 다음 질문이 동시에 화면에 표시되어 모바일 환경에서 사용성이 떨어지는 문제가 있었습니다.

### 해결

- 질문과 피드백 흐름 개선
- 답변 확인 후 다음 질문 생성
- 모바일 환경을 고려한 UX 개선

---

### 공통 Seed 구조 개선

### 문제

언어별 및 레벨별 Seed 파일을 각각 관리할 경우 유지보수가 어려워질 가능성이 있었습니다.

### 해결

- 공통 AI Seed 스크립트만 사용하도록 구조 변경
- 언어 및 레벨 파라미터 기반 생성 구조 적용
- 공통 Prompt를 활용한 데이터 생성 구조 설계
