# DEX Institution Admin (BEX Admin) — Web3 거래소 운영 관리자

## 프로젝트 소개

기관 및 운영자를 위한 Web3 기반 DEX(탈중앙 거래소) 관리자 플랫폼입니다.

Swap Pool 관리, Liquidity 운영, 수수료 Claim, Token Transfer, 거래 현황 조회 등 DEX 운영에 필요한 기능을 통합한 Admin 서비스입니다.

Next.js App Router 기반으로 개발되었으며, Cosmos 계열 체인과 EVM 계열 체인을 동시에 지원하는 구조로 설계되었습니다.

WalletConnect(Reown AppKit), CosmosJS, Wagmi 등을 활용하여 실제 블록체인 서명 및 트랜잭션 브로드캐스트 기능까지 구현했습니다.

## 트러블 슈팅 경험

### WalletConnect 환경별 연결 문제 해결

### 문제

로컬 환경에서는 정상 동작하지만 배포 환경에서 Wallet Binding 실패 발생

### 원인

- 프로젝트 ID 및 도메인 설정 문제
- WalletConnect 허용 도메인 불일치

### 해결

- 운영 도메인 등록
- 환경별 Project ID 관리 구조 개선

---

### Next.js 환경에서 API 보안 처리

### 문제

클라이언트에서 직접 외부 API를 호출하면 보안상 문제가 발생

### 해결

- Next.js Route Handler(API Route) 활용
- 서버 사이드에서 외부 API 호출 구조 설계
- Token 및 민감 정보 서버 내부 처리
