# GuruScan — Web3 Blockchain Explorer

## 프로젝트 개요

Cosmos 기반 블록체인 네트워크의 데이터를 조회하고 분석할 수 있는 Blockchain Explorer 서비스입니다.

Block, Transaction, Validator, Governance, IBC, Bridge, Smart Contract, Account 등의 온체인 데이터를 실시간으로 조회할 수 있도록 구현했으며, 운영 상태를 시각적으로 확인할 수 있는 Dashboard 형태의 UI를 구축했습니다.

Next.js App Router 기반으로 개발되었으며, 실시간 데이터 처리와 대량 데이터 렌더링 최적화에 중점을 두고 개발한 프로젝트입니다.

## 트러블 슈팅

### Chart 렌더링 오류 해결

### 문제

실시간 데이터 업데이트 과정에서 ApexCharts 렌더링 오류 및 메모리 증가 발생

### 해결

- 데이터 검증 로직 추가
- NaN 데이터 예외 처리
- 렌더링 최소화 구조 적용

---

### Sticky Layout 및 반응형 이슈 해결

### 문제

복잡한 Layout 구조에서 Sticky Header 및 반응형 동작이 정상적으로 동작하지 않는 문제 발생

### 해결

- overflow 구조 개선
- min-width 및 flex/grid 구조 최적화
- 레이아웃 계층 구조 재설계
