# GXS Additional Platform — Web3 서비스 및 인터랙션 플랫폼

## 프로젝트 개요

Web3 기반 서비스 및 블록체인 생태계를 사용자에게 직관적으로 전달하기 위한 플랫폼 프로젝트입니다.

브랜드 아이덴티티와 사용자 경험(UI/UX)을 중심으로 설계되었으며, 다양한 인터랙션 요소와 반응형 UI를 활용해 사용자 친화적인 웹 환경을 구축한 것이 특징입니다.

Vue 3 기반으로 개발되었으며, 퍼포먼스와 유지보수성을 고려한 컴포넌트 구조 설계 및 반응형 최적화를 함께 진행했습니다.

---

## 담당 역할

- Front-End 개발 전반 담당
- 메인 랜딩 페이지 및 인터랙션 UI 개발
- 반응형 레이아웃 구현
- 공통 컴포넌트 설계 및 구조화
- 애니메이션 및 사용자 경험 개선
- 퍼포먼스 최적화

## 주요 기능

### 1. Web3 서비스 소개 페이지

플랫폼의 주요 서비스 및 블록체인 관련 기능을 사용자에게 직관적으로 전달할 수 있도록 랜딩 페이지를 구축했습니다.

### 구현 포인트

- 브랜드 중심의 UI 설계
- Section 단위 Layout 구성
- 자연스러운 스크롤 흐름 구성
- 사용자 경험 중심 인터랙션 구현

---

### 2. SVG 및 인터랙션 애니메이션

정적인 페이지가 아닌 브랜드 경험을 강화하기 위한 인터랙션 요소를 구현했습니다.

### 구현 기능

- Scroll 기반 Animation
- Hover Interaction
- SVG Motion Effect
- Transition 기반 사용자 인터랙션

### 구현 포인트

- 브라우저 호환성 고려
- 성능 중심 애니메이션 처리
- 모바일 환경 최적화
- GPU 가속 활용

---

### 3. 반응형 UI 및 Layout 설계

Desktop / Tablet / Mobile 환경에 대응할 수 있도록 반응형 UI를 구현했습니다.

### 구현 포인트

- Grid/Flex 기반 Layout 구성
- 모바일 Navigation 처리
- 반응형 Typography 적용
- 화면 크기별 Layout 최적화

---

### 4. 공통 컴포넌트 구조화

프로젝트 내 반복적으로 사용되는 UI를 공통 컴포넌트로 분리하여 유지보수성과 재사용성을 높였습니다.

### 구현 컴포넌트

- Header
- Navigation
- Button
- Card
- Footer
- Section Wrapper
- Modal

### 구현 포인트

- Props 기반 확장 구조 설계
- 공통 스타일 관리
- 재사용 가능한 컴포넌트 설계

- 모바일 환경 안정성 개선

### 반응형 Layout 최적화

복잡한 Layout 구조에서 발생하는 overflow 및 UI 깨짐 현상을 개선했습니다.

## 트러블 슈팅

### 문제

### 문제

- Animation 구조 단순화
- GPU 가속 활용
- 렌더링 최소화 구조 적용
