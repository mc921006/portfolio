# Gurufin Corporate Website — Web3 기업 홈페이지

## 프로젝트 개요

Web3 및 블록체인 기반 금융 서비스를 소개하는 Corporate Website 프로젝트입니다.

기업 소개, 서비스 설명, 기술 비전, 파트너십, 네트워크 구조 등을 사용자에게 직관적으로 전달할 수 있도록 설계했으며, 다양한 SVG 애니메이션과 인터랙션 요소를 활용하여 브랜드 아이덴티티를 표현한 것이 특징입니다.

반응형 환경을 고려하여 Desktop / Tablet / Mobile 전 환경에서 자연스럽게 동작하도록 구현했으며, SEO 및 퍼포먼스 최적화까지 함께 진행했습니다.

---

## 주요 기능

기업의 서비스 및 기술 비전을 소개하는 메인 랜딩 페이지를 구축했습니다.

### 제공 기능

- 서비스 소개
- 블록체인 네트워크 설명
- 파트너사 소개
- 기술 구조 시각화

### 구현 포인트

- 브랜드 중심 UI 설계
- 반응형 레이아웃 구성
- Section 단위 인터랙션 처리
- 자연스러운 스크롤 흐름 구현

### 구현 포인트

- 반응형 환경 대응

## 트러블 슈팅

### SVG Import 및 빌드 오류 해결

### 문제

프로젝트별로 SVG import 방식 차이로 인해 빌드 오류 발생

### 해결

- @svgr/webpack 설정 적용
- SVG Component 구조 통일
- 빌드 환경별 설정 최적화

---

### 반응형 Chart/Layout 깨짐 현상 해결

### 문제

특정 해상도에서 Grid/Flex 구조가 깨지며 UI가 비정상적으로 렌더링되는 문제 발생

### 해결

- min-width: 0 적용
- Grid/Flex 구조 재설계
- 반응형 기준 재정의
