# WWW-LAND — Web3 랜딩 및 브랜드 플랫폼

## 프로젝트 개요

Web3 서비스 및 블록체인 생태계를 사용자에게 소개하기 위한 브랜드 중심 랜딩 플랫폼 프로젝트입니다.

서비스 소개, 기술 비전, 브랜드 아이덴티티, 네트워크 구조 등을 시각적으로 전달할 수 있도록 구성했으며, 다양한 인터랙션 요소와 애니메이션을 활용해 사용자 경험(UI/UX)을 강화한 것이 특징입니다.

Vue 3 및 Vite 기반으로 개발되었으며, 반응형 UI와 퍼포먼스를 고려한 구조 설계에 중점을 두고 개발했습니다.

---

## 담당 역할

- Front-End 개발 전반 담당
- 메인 랜딩 페이지 UI 개발
- SVG 및 인터랙션 애니메이션 구현
- 반응형 Layout 설계
- 공통 컴포넌트 구조화
- 사용자 경험(UI/UX) 개선
- 퍼포먼스 최적화

## 주요 기능

### 1. 브랜드 랜딩 페이지 구축

브랜드 및 Web3 서비스를 사용자에게 직관적으로 전달할 수 있는 랜딩 페이지를 구축했습니다.

### 제공 기능

- 서비스 소개
- 브랜드 비전 안내
- 네트워크 및 기술 설명
- 사용자 인터랙션 기반 UI
- Contact 및 정보 제공

### 구현 포인트

- 브랜드 중심 UI 설계
- Section 기반 Layout 구성
- 자연스러운 스크롤 흐름 구현
- 사용자 경험 중심 인터랙션 구성

---

### 2. SVG 및 인터랙션 애니메이션 구현

정적인 페이지가 아닌 사용자 몰입감을 높일 수 있도록 다양한 애니메이션 및 인터랙션 요소를 구현했습니다.

### 구현 기능

- Scroll 기반 Animation
- Hover Interaction
- SVG Motion Effect
- Transition Animation

### 구현 포인트

- SVG Component 구조화
- 성능을 고려한 Animation 처리
- 브라우저 호환성 고려
- 모바일 환경 최적화

---

### 3. 반응형 UI 및 Layout 설계

Desktop / Tablet / Mobile 환경에 대응할 수 있도록 반응형 UI를 구현했습니다.

### 구현 포인트

- Grid/Flex 기반 Layout 구성
- 반응형 Navigation 처리
- 모바일 Fullscreen Menu 구현
- 화면 크기별 Layout 최적화

---

### 4. 공통 컴포넌트 구조화

프로젝트 내 반복적으로 사용되는 UI 요소를 공통 컴포넌트화하여 유지보수성과 재사용성을 높였습니다.

### 구현 컴포넌트

- Header
- Navigation
- Button
- Card
- Section Wrapper
- Modal
- Footer

### 구현 포인트

- 재사용 가능한 구조 설계
- Props 기반 확장성 고려
- 공통 스타일 구조화

## 성능 최적화 경험

### 애니메이션 렌더링 최적화

다수의 인터랙션 및 애니메이션 요소 사용으로 인해 발생할 수 있는 렌더링 부하를 개선했습니다.

### 개선 내용

- CSS 기반 Animation 우선 적용
- 불필요한 Re-render 최소화
- SVG 및 이미지 최적화
- GPU 가속 활용

### 결과

- 스크롤 성능 향상
- 모바일 환경 안정성 개선
- 사용자 경험 향상

---

### 반응형 Layout 최적화

복잡한 Layout 구조에서 발생하는 overflow 및 UI 깨짐 현상을 개선했습니다.

### 개선 내용

- Grid/Flex 구조 개선
- min-width 및 overflow 최적화
- Layout 구조 재설계

---

## 트러블 슈팅

### 반응형 UI 깨짐 문제 해결

### 문제

특정 해상도에서 UI 요소가 겹치거나 Layout이 비정상적으로 동작하는 문제 발생

### 해결

- Flex/Grid 구조 재설계
- 반응형 기준 재정의
- overflow 및 width 구조 개선

---

### 애니메이션 성능 저하 문제 해결

### 문제

다수의 애니메이션 요소 사용으로 일부 환경에서 렌더링 성능 저하 발생

### 해결

- Animation 구조 단순화
- GPU 가속 활용
- 렌더링 최소화 구조 적용
