# DEX Admin FE — Web3 DEX 운영 관리자 플랫폼

## 프로젝트 개요

Web3 기반 DEX(탈중앙 거래소) 운영을 위한 관리자(Admin) 플랫폼입니다.

운영자가 거래소 상태를 확인하고, 유동성 및 거래 데이터를 관리하며, WalletConnect 기반 지갑 연결을 통해 실제 블록체인 트랜잭션을 수행할 수 있도록 구축한 프로젝트입니다.

Next.js App Router 기반으로 개발되었으며, Cosmos 및 EVM 계열 네트워크를 동시에 지원하는 멀티체인 구조로 설계되었습니다.

## 트러블 슈팅

### WalletConnect 배포 환경 인증 문제

### 문제

로컬 환경에서는 정상 동작하지만 운영 환경에서 Wallet Binding 실패 발생

### 원인

- WalletConnect 프로젝트 설정 및 허용 도메인 불일치

### 해결

- 운영 도메인 등록
- 환경별 Project ID 관리 구조 개선

---

### Chart 렌더링 및 메모리 증가 문제

### 문제

실시간 데이터 업데이트 과정에서 Chart 렌더링이 반복되며 메모리 사용량 증가

### 해결

- 데이터 중복 업데이트 방지
- 렌더링 최소화 구조 적용
- 상태 변경 흐름 개선
