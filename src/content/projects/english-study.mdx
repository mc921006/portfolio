# English Study — AI 기반 영어·베트남어 학습 서비스

## 프로젝트 개요

AI를 활용하여 영어와 베트남어를 보다 효율적으로 학습할 수 있도록 개발한 개인 학습 서비스입니다.

단어, 문법, 회화 학습을 하나의 서비스에서 제공하며, AI를 통해 회화 질문 및 피드백을 생성하고, TTS(Text-to-Speech)를 활용하여 원어민 발음을 들으며 학습할 수 있도록 구현했습니다.

영어는 CEFR(A1~C2) 레벨 기반으로 학습할 수 있으며, 베트남어는 실생활에서 자주 사용하는 표현을 중심으로 학습할 수 있도록 구성했습니다.

반응형 UI를 적용하여 Desktop과 Mobile 환경 모두에서 사용할 수 있도록 구현했으며, Supabase와 AI를 활용하여 학습 데이터를 관리할 수 있는 구조를 설계했습니다.

## 주요 기능

### 1. 단어 학습

영어와 베트남어 단어를 레벨별로 학습할 수 있도록 구현했습니다.

### 제공 기능

- 영어 CEFR(A1~C2) 학습
- 베트남어 기초 단어 학습
- 단어 의미 제공
- 예문 제공
- 발음(TTS) 지원

### 구현 포인트

- 언어별 데이터 분리
- 레벨 기반 단어 조회
- 공통 구조를 활용한 데이터 관리
- 재사용 가능한 학습 UI 구성

---

### 2. AI 기반 회화 학습

AI를 활용하여 사용자의 학습 주제에 맞는 회화 질문을 생성하고, 답변에 대한 피드백을 제공하도록 구현했습니다.

### 제공 기능

- 주제별 회화 질문 생성
- 사용자 답변 입력
- AI 답변 평가
- 다음 질문 자동 생성

### 구현 포인트

- 프롬프트 기반 질문 생성
- 답변 품질 피드백 제공
- 자연스러운 학습 흐름 구성
- 반복 학습이 가능한 구조 설계

---

### 3. 문법 학습

영어 문법을 CEFR 레벨에 맞춰 학습할 수 있도록 구성했습니다.

### 제공 기능

- 레벨별 문법 학습
- 문법 설명
- 예문 제공

### 구현 포인트

- 레벨별 콘텐츠 분리
- 일관된 UI 구조
- 학습 중심 화면 구성

---

### 4. TTS 발음 기능

단어의 발음을 직접 들으며 학습할 수 있도록 TTS 기능을 구현했습니다.

### 구현 기능

- 단어 발음 재생
- 언어별 발음 지원
- 직관적인 발음 버튼 제공

### 구현 포인트

- 학습 흐름을 방해하지 않는 UI
- 발음 기능과 단어 카드 분리
- 사용자 경험을 고려한 인터랙션

---

### 5. 반응형 UI

Desktop과 Mobile 환경 모두에서 사용할 수 있도록 반응형 UI를 구현했습니다.

### 구현 포인트

## 트러블 슈팅

### AI 생성 데이터 중복 문제

### 문제

AI가 생성한 단어가 기존 데이터와 중복되어 저장되는 문제가 발생했습니다.

### 해결

- 영어 전체(`language = 'en'`) 기준으로 중복 체크
- 기존 단어는 유지하고 새로운 단어만 저장
- 공통 AI Seed 스크립트에서 중복 검증 로직 추가

---

### AI 회화 UX 개선

### 문제

답변 평가와 다음 질문이 동시에 화면에 표시되어 모바일 환경에서 사용성이 떨어지는 문제가 있었습니다.

### 해결

- 질문과 피드백 흐름 개선
- 답변 확인 후 다음 질문이 생성되도록 UX 변경
- 모바일 환경을 고려한 인터페이스 개선

---

### 공통 Seed 구조 개선

### 문제

레벨마다 별도의 Seed 파일을 관리할 경우 유지보수가 어려워질 가능성이 있었습니다.

### 해결

- 공통 AI 자동 생성 Seed 스크립트만 사용하도록 구조 변경
- 레벨별 하드코딩 제거
- 재사용 가능한 생성 구조 설계
