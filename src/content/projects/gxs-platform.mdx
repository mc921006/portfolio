# GXS Platform — Web3 서비스 플랫폼

## 프로젝트 개요

Web3 기반 서비스 및 블록체인 생태계를 사용자에게 제공하기 위한 플랫폼 프로젝트입니다.

서비스 소개, 네트워크 정보, 블록체인 기능 안내, 사용자 인터랙션 등을 중심으로 구성된 플랫폼 형태의 웹 서비스이며, 다양한 애니메이션 및 반응형 UI를 활용하여 사용자 경험을 강화한 것이 특징입니다.

Vue 3 기반으로 개발되었으며, 사용자 친화적인 인터페이스와 브랜드 경험 중심의 UI/UX 설계에 중점을 두고 개발했습니다.

---

## 담당 역할

- Front-End 개발 전반 담당
- 메인 페이지 및 서비스 UI 개발
- 반응형 레이아웃 구현
- 인터랙션 및 애니메이션 구현
- 공통 컴포넌트 구조화
- 사용자 경험(UI/UX) 개선
- 퍼포먼스 최적화

## 주요 기능

### 1. 메인 랜딩 및 서비스 소개 페이지

플랫폼의 서비스 및 기능을 사용자에게 직관적으로 전달할 수 있도록 메인 랜딩 페이지를 구축했습니다.

### 제공 기능

- 서비스 소개
- 플랫폼 기능 안내
- 네트워크 정보 제공
- 브랜드 비주얼 구성
- 사용자 인터랙션 제공

### 구현 포인트

- Section 기반 Layout 구성
- 스크롤 흐름 중심 UI 설계
- 반응형 레이아웃 처리
- 사용자 경험 중심 인터랙션 구성

---

### 2. 인터랙션 및 애니메이션 구현

정적인 페이지가 아닌 사용자 경험을 강화하기 위한 인터랙션 요소를 구현했습니다.

### 구현 기능

- Scroll 기반 애니메이션
- Hover 인터랙션
- SVG 기반 효과
- Transition 및 Motion 효과

### 구현 포인트

- 성능을 고려한 애니메이션 처리
- 브라우저 호환성 고려
- 모바일 환경 최적화
- 자연스러운 사용자 흐름 구성

---

### 3. 반응형 UI 구현

Desktop / Tablet / Mobile 환경에 대응할 수 있도록 반응형 UI를 구현했습니다.

### 구현 포인트

- Grid/Flex 기반 Layout
- 모바일 Navigation 처리
- 반응형 Typography 구성
- 화면 크기별 UI 최적화

---

### 4. 공통 컴포넌트 구조화

반복적으로 사용되는 UI 요소를 공통 컴포넌트로 분리하여 유지보수성과 재사용성을 높였습니다.

### 구현 컴포넌트

- Header
- Navigation
- Button
- Card
- Section Wrapper
- Modal
- Footer

### 구현 포인트

- Props 기반 재사용 구조 설계
- 공통 스타일 구조화
- 확장 가능한 컴포넌트 설계

## 성능 최적화 경험

### 애니메이션 렌더링 최적화

다수의 애니메이션 및 인터랙션 요소 사용으로 인해 렌더링 부하가 발생할 수 있어 최적화를 진행했습니다.

### 개선 내용

- CSS 기반 애니메이션 우선 적용
- 불필요한 Re-render 최소화
- GPU 가속 활용
- SVG 및 이미지 최적화

### 결과

- 스크롤 성능 향상
- 모바일 환경 안정성 개선
- 사용자 경험 향상

---

### 반응형 Layout 최적화

복잡한 Layout 구조에서 발생하는 overflow 및 UI 깨짐 현상을 개선했습니다.

### 개선 내용

- Grid/Flex 구조 최적화
- min-width 및 overflow 구조 개선
- 반응형 기준 재설계

---

## 트러블 슈팅

### 반응형 UI 깨짐 현상 해결

### 문제

특정 해상도에서 Layout이 깨지고 요소가 겹치는 문제가 발생

### 해결

- Layout 구조 재설계
- Flex/Grid 구조 개선
- 화면 크기별 스타일 최적화

---

### 애니메이션 성능 저하 문제 해결

### 문제

다수의 애니메이션 요소로 인해 일부 환경에서 렌더링 성능 저하 발생

### 해결

- 애니메이션 구조 단순화
- GPU 가속 활용
- 렌더링 최소화 구조 적용
